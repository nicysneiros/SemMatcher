The Edit Operation Dialog Box

For merge and moving frames modes, the Edit Operation dialog box allows you to edit a suggested operation on the  To Do list at the Suggestions tab. You can access this dialog box by clicking the View icon at the top right of the To Do list.

The following things can be edited from this dialog box:

  • The operations menu at the top allows you to choose a different operation from the list of operations available for the current mode. For more information, see Available PROMPT Operations.
  • The ontology menu(s) allow you to choose the ontologie(s) for the argument, for example, to set the source ontology for a copy operation in merge mode.
  • The frame entry bar allows you to choose the frame(s) you want as arguments to the operation. You cannot type a name in this bar; you must select it using the Add button.
  • The option check boxes allow you to change the options for the operation. This is probably the most frequent reason for editing an existing suggestion. For more information about options, see Available PROMPT Operations.

When you edit a suggestion, the old suggestion is deleted from the list and replaced with the new one. For this reason, you would seldom want to change the operation or the arguments. (For a merge operation, you might want to change one of the arguments, leaving the other one untouched.) If you want to create a new suggestion, use the Create Operation Dialog Box, accessed using the Create button at the upper right of the To Do list. If you want to delete a suggestion, use the Delete button.
