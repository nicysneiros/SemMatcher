Removing Suffixes

If you are merging ontologies, each time you copy a frame from one of the source ontologies to the working ontology,  PROMPT attaches a suffix to the frame name. The suffix, which indicates where the frame came from, is set in the Alias entry bar in the initialization dialog box. Although suffixes are useful during the merge process, you usually will want to remove them in the final version of the ontology.

To remove all suffixes:

  1. Select Remove Suffixes from the PROMPT menu.
    The suffixes are removed. If there are any frames that have the same name except for the suffix, Protégé appends a number to one of them, such as _1.

When you merge frames, the merged frame does not have the suffix.
