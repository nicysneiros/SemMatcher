Installing PROMPT

Before you can use PROMPT for the first time, you must install it in the plugins folder. This adds the PROMPT functionality to Protégé. Once PROMPT has been installed correctly, you can use it in any project by enabling the Prompt Tab in your target project.. You do not have to reinstall it for a new project. 

To install PROMPT:

  1. Download the prompt file from  the PROMPT page in the Protégé Plugins library (http://protege.stanford.edu/plugins).  
  2. Unzip the file prompt.zip, placing the resulting jar file in the plugins directory. You can either unzip it directly to plugins, or move it in later.

If prompt.jar has been installed, the Configure dialog box (opened by choosing Configure on the Project menu) will display a PromptTab option. If PromptTab does not appear in the Project | Configure dialog box, check the following:
  1. Make sure that the file prompt.jar is unzipped and installed in the plugins subdirectory of the Protégé directory.

  2. If you have more than one copy of Protégé installed, make sure that you are running the copy for which you installed the plug-in.
