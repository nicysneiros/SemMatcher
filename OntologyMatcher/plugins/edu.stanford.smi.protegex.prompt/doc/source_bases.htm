The Source Knowledge Bases Window

The Source Knowledge Bases window shows you one of the following:

  • the argument(s) of the currently highlighted suggestion (at The Suggestions Tab) as those arguments appear in the original ontologie(s) that you incorporated into your working project. 
  • the source frames of the currently highlighted frame in the Result Window.

You can change the view by clicking the appropriate tab, but you cannot change the selection directly in this window. 

The exact contents of the window depend on the mode in which you are working:

  • If you are merging ontologies, the ontology selected first during initialization is on the left. 
  • If you are extracting or moving frames, you will see only one ontology.
  • If you are comparing two versions of the same ontology, the old ontology is on the left. 

To view the Source Knowledge Bases window, do one of the following:

  • From the Suggestions Tab, click on the Source icon in the Suggestions tab. In this case, the argument(s) to whichever suggestion is highlighted in the Suggestions tab are displayed and highlighted in the Source Knowledge Bases window.
  • From the Result Window, click on the Source icon in the Result window. In this case, the source frame(s) to the frame that is highlighted in the Result window are displayed and highlighted in the Source Knowledge Bases window.

To change the selected frame(s), go to the Suggestions Tab and click on a different suggestion in the ToDo list, or select a different frame in the Result Window.

Source Buttons

Buttons at the top of each pane allow you to view information about the highlighted frame:

V(iew) button: Click this button to open the Class, Slot, or Instance Form for the highlighted frame. Because you are viewing the original ontology, you cannot edit the frame from this version of the form. However, you may be able to access additional information, such as viewing the slots of a selected class.
Back-references button: Click this button to view all the objects in the original ontology that reference the highlighted frame. 

Source Icons

Special icons to the right of a frame indicate its status:

The frame has been mapped (merge or extract modes). A mapped frame has been merged or copied into the working project. It has not been altered in the original ontology.
