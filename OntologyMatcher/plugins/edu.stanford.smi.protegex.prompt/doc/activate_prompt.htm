Enabling the Prompt Tab

Before you can use PROMPT, you must enable and initialize the Prompt tab. You need to enable the tab in your target project. 

  • If you are merging two ontologies, your target project is an empty project which will later contain the merged ontology. 
  • If you are extracting a part of an ontology, your target project is usually a new project into which you will extract the part. You can also use a project that already has an ontology in it: in this case, the part that you extract will be added to that ontology. 
  • If you are moving frames from an included project to the including one, your target project is the including one, that is, the larger project that references the included project as a part of the overarching ontology..
  • If you are comparing different versions of the same ontology, your target project is an empty project.

To enable the tab:

  1. Open or create the project where you wish to use PROMPT. For example, if you are planning to merge two ontologies, or to extract part of an ontology, create a new project that will contain the merged or extracted ontology. If you are moving frames from an included to an including project, open the including project.
  2. If you have created a new project, and you have not yet saved it, it is best to save it before you enable PROMPT. This will tell PROMPT where to store its log. 
  3. Select Project | Configure from the menu.
    The Configure dialog box is displayed.
  4. Check the PromptTab.
     
    NOTE: If the PromptTab does not appear in the Configure dialog box, check the following:
    1. Make sure that the file prompt.jar is unzipped and installed in the plugins subdirectory of the Protégé directory.

    2. If you have more than one copy of Protégé installed, make sure that you are running the copy for which you installed the plug-in.

  5. If you wish PROMPT to use its color coding on all the standard Protégé tabs, use the Up arrow in the top-left corner of the Configure dialog to move the PromptTab to the top of the list. If you do not place PromptTab at the top, color-coding will only be visible at the PromptTab itself.
  6. Click OK to close the Configure dialog box. 
    You will be shown the Prompt tab, which at this point is simply one large button. Click anywhere in this area to initialize PROMPT. See Initializing PROMPT for more information.
