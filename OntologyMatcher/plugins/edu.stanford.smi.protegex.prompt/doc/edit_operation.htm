Editing a Suggested Operation

For merge and moving frames modes, suggested operations are shown on the To Do list at the Suggestions tab. You can edit operations on this list, for example, to change the options, as follows:

  1. Select the operation you want to change by clicking on it. You can only edit one operation at a time.
  2. Click the View button at the top right of the To Do list to open the Edit Operation Dialog Box.
  3. To change the operation, select a new operation from the operations menu at the top. Changing the operation will remove the original operation from the To Do list, replacing it with the operation you create. 
  4. To change an argument for the operation, click the Add button. The Select Frame dialog box is displayed.

    Click on the desired frame to highlight it and click OK.
    Note: If you want to select an argument from a different ontology, first change the ontology on the menu immediately above the Argument entry box.
  5. To change the options for the arguments, click the check boxes for the desired options. The available options depend on the operation you have chosen. See PROMPT Operations for more information.
  6. To perform the operation immediately, click Do It.  In some cases, you may receive a warning dialog box. See Performing an Operation for more information.
    When you perform an operation, the To Do list at the Suggestions Tab, the Conflicts Tab, and the Result Window are all updated accordingly. If you view any of the other tabs, such as the Classes tab, the frame incorporated by your operation will be visible.
  7. To add the operation to the To Do list at the Suggestions Tab, click ToDo list. The operation will be listed on the To Do list, with the name of the operation in bold to indicate that it has been created or modified. You can perform this operation later by going to the Suggestions Tab, highlighting the operation, and clicking Do It.
