Viewing Source Knowledge Bases

When you are working with the Suggestions Tab or the Result Window, you can open the Source Knowledge Bases window to see the status of the highlighted frame(s) in their original ontologie(s). 

To view the source knowledge base information from the Suggestions tab:

  1. At the Suggestions Tab, highlight the suggestion you want to view by clicking on it. You can only view information for one row at a time. If more than one suggestion is highlighted, the top suggestion will be displayed.
  2. Click the Source button at the upper right of the ToDo list.
    The Source Knowledge Bases window is displayed. The argument(s) to the current operation are highlighted in the appropriate ontologie(s).
  3. To change the view, click on a different suggestion in the Suggestions Tab.

To view the source knowledge base information from the Result Window:

  1. At the Result Window, highlight the frame whose sources you want to view. This should be a frame which was created via a PROMPT operation from one or more frames in the original ontologie(s). You can only view source information for one result frame at a time. 
  2. Click the Source button at the upper right of the Result Window.
    The Source Knowledge Bases window displays the original ontologie(s). The source frame(s) to the selected frame are highlighted in the appropriate ontologie(s). For example, if the frame was copied from an ontology, the original frame will be highlighted in that ontology.
  3. To change the view, click on a different frame in the Result Window.

To close the window, click on the upper right corner.
