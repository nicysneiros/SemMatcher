Setting Up PROMPT

This topic summarizes the steps you need to follow before you can use PROMPT. These steps are described more fully in the following topics:

  1. Install the file prompt.jar in the plugins directory. 
    This adds the PROMPT functionality to Protégé . Once PROMPT is installed correctly, you can use it in any project. You do not have to reinstall it for a new project.
  2. Activate the Prompt Tab in your target project. 
    This configures Protégé to display the Prompt Tab. Often, your working project will be an empty project created specifically to use the PROMPT functionality. You must activate the Prompt Tab in each project where you want to use PROMPT.
  3. Initialize PROMPT in your target project. 
    At this point, you choose the ontologies you working with and the PROMPT functionality you want to use. Protégé performs the appropriate initial heuristics and sets up the Prompt tab window for the files you have chosen. You must do this for each project where you want to use PROMPT.
