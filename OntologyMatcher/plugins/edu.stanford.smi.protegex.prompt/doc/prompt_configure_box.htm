The PROMPT Configuration Dialog Box

The PROMPT Configuration Dialog Box allows you to change PROMPT's behaviour. To view this dialog box, select Configure from the Prompt menu.

You can change the following settings:

  • Consider inherited slots: When you copy or merge a class, PROMPT copies any slots that are directly attached to the class. When Consider inherited slots is selected, PROMPT also copies the slots that are inherited by the class.
  • Case-sensitive ontology: When selected, frame names that are the same except for capitalization are considered to be different; deselect to have such frame names considered to be identical. 
  • Set preferred ontology: When you are merging ontologies, allows you to set one of the ontologies as the default whenever you are merging two frames with different names, documentation strings, or roles (that is, different own slot values). If not set, PROMPT asks you to choose the value for the merged frame. 

Note: PROMPT configuration can only take place after you have enabled PROMPT and chosen your source ontologies. To learn about enable PROMPT by selecting Configure from the Project menu, see Enabling the Prompt Tab.
