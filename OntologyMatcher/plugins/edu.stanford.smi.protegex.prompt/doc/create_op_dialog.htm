The Create Operation Dialog

The Create Operation dialog allows you to create a new prompt operation and either perform the operation or add it to the To Do list in the Suggestions Tab.

You can view the Create Operation dialog by clicking the Create button in the ToDo list at the Suggestions Tab.

Note: A similar dialog appears at the bottom of the New Operations Tab. However, the New Operations Tab has additional functionality that allows you to use the information in the top half to load frames.

The dialog box contains the following sections:

  • A menu which allows you to choose the operation you want to create. Operations may involve classes, slots, or instances. Available operations depend on the mode (merge, extract, move from included, or compare) in which you are working. See PROMPT Operations for more information.
  • One or two entry bars where you can choose the class(es), slot(s), or instance(s) as arguments to the operation. Merge operations have two arguments; all other operations have only one argument. To select or change an argument, click the Add button. 
  • Check boxes that allow you to select the options for the operation. Options are described in PROMPT Operations.
  • Buttons that allow you to Do It, which performs the operation immediately; or ToDo list, which adds the operation to the To Do list at the Suggestions Tab.
