The Prompt Tab

The Prompt tab allows you to view the ontologies you are merging, including, extracting, or comparing, and to choose the operations to perform on all or part of these ontologies. 

To view the Prompt Tab in your project, you must have PROMPT installed in your Protégé plugins folder and you must activate the Prompt Tab in your project. Before the Prompt Tab is initialized, it is just one giant button: 

Clicking anywhere in this area will start the initialization process, which allows you to choose which ontologies you want to work with, and the actions you will use. 

Once the Prompt Tab has been initialized, it displays a number of subtabs related to the PROMPT mode you have chosen. 

For merge, extract, and moving frames modes, an interface similar to the following will appear:

The area on the left-hand side has two or three subtabs:

  • The Suggestions Tab (merge and moving frames modes only) : Shows a list of frames from the sources ontologie(s) and a suggested action for each frame shown. The Suggestions list is updated according to the operations you perform. A pane at the bottom displays the Protégé heuristics for the suggestion. Clicking Do It performs the suggested action for every highlighted frame.
  • The Conflicts Tab (optional): Shows a list of "conflicts" that need to be resolved to make the project logically complete. For example, if a class has a slot that references a class that currently does not exist in the working project, this will show up as a conflict.  The Conflicts list is updated according to the operations you perform. A pane at the bottom gives suggested solutions for the conflict. When there are conflicts, the title of the tab is red .
  • The New Operations Tab: Shows the source ontologie(s) and allows you to create new PROMPT operations. You can perform the operations from this pane, or add them to the To Do list at the Suggestions Tab.

The Result Window on the right-hand side allows you to view the current state of your working ontology. You can view classes, slots, or instances. This information is part of the view you would get by clicking the Class, Slot, or Instances Tab. It is duplicated here for convenience.

See the individual pages for more information on each area.

If you are using compare mode, you will see the image table instead. See The Image Table for more information.
