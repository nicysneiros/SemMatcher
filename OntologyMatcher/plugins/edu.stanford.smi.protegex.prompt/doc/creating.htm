Creating an Operation

You can create a PROMPT operation in one of two ways:

  • By going to the New Operations tab
  • By opening the Create Operations dialog box by clicking on the Create button at the Suggestions Tab.

In either case, once the operation has been set up, you can either perform it immediately by clicking the Do It button, or you can add it to the ToDo list at the Suggestions Tab.

To create an operation:

  1. Make sure the Prompt tab is selected.
  2. Click the New Operations tab
    OR
    Click the Suggestions tab and then click the Create button at the top right hand side of the ToDo list.
  3. Select the operation you want from the operations menu. See PROMPT Operations for a list of operations.

    If you have selected a merge operation you will see two arguments. (Arguments are the classes, slots, or instances on which you operate.) If you have selected a copy, move, or remove parent operation, you will see only one argument.
  4. To change the ontolog(ies) for the argument, make a selection from the ontology menu(s). For instance, if you are merging two ontologies, and you have chosen a copy operation, you can use this menu to select the ontology from which you want to copy a frame. Or, if you want to merge frames that are now in the working ontology, select the working ontology from this menu. 
  5. To select an argument for the operation, click the Add button. 

    The Select Frame dialog box is displayed.

    Click on the desired frame to highlight it and click OK.
    OR
    If you are using the New Operations tab, you can highlight the frame you want in the Source Knowledge Bases area at the top and click the Load button.
  6. If there is a second argument, click the second Add button and select the desired frame from the dialog box
    OR
    In the New Operations tab, highlight the frame you want in the Source Knowledge Bases area at the top and click the Load button.
  7. To set the options for the arguments, click the check boxes for the desired options. The available options depend on the operation you have chosen. See PROMPT Operations for more information.
  8. To perform the operation immediately, click Do It. In some cases, you may receive a warning dialog box. See Performing an Operation for more information.
    When you perform an operation, the To Do list at the Suggestions Tab, the Conflicts Tab, and the Result Window are all updated accordingly. If you view any of the other tabs, such as the Classes tab, the frame incorporated by your operation will be visible.
  9. For merge and moving frames modes, to add the operation to the To Do list at the Suggestions Tab, click ToDo list. The operation will be listed on the To Do list, with the name of the operation in bold to indicate that it has been created or modified. You can perform this operation later by going to the Suggestions Tab, highlighting the operation, and clicking Do It.
